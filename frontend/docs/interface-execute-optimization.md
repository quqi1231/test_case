# InterfaceExecute.vue 功能优化 PRD

## 1. 项目概述
- **项目名称**: 接口自动化测试平台 - 页面优化
- **目标用户**: 测试工程师、开发人员
- **优化目标**: 修复现有问题，提升用户体验

## 2. 需求清单

### 2.1 删除对比模式
- **问题**: 对比模式功能不实用，且容易导致错误
- **方案**: 移除对比模式相关代码

### 2.2 修复编辑按钮布局重叠
- **问题**: 编辑按钮和链按钮重叠
- **方案**: 调整表格操作列宽度，或换行显示

### 2.3 编辑功能优化
- **问题**: 点击编辑弹出对话框，不够直观
- **方案**: 点击编辑后，直接将接口内容填充到左侧请求配置区域，类似"点击接口列表自动填充"

### 2.4 导出和报告优化
- **问题**: 导出和报告默认操作全部数据
- **方案**: 
  - 导出：优先导出选中的接口，无选中时导出当前搜索结果
  - 报告：生成当前搜索结果/选中接口的报告

## 3. 验收标准
- [ ] 对比模式按钮已移除
- [ ] 编辑按钮和链按钮不再重叠
- [ ] 点击编辑后接口信息填充到左侧配置区
- [ ] 导出功能支持选中导出
- [ ] 报告功能支持选中生成

## 4. 技术方案
- 修改 InterfaceExecute.vue 模板和脚本
- 保持与后端 API 兼容
- 不破坏现有功能
